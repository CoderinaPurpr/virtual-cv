# Rudi van Vuuren — Virtual CV

A responsive personal portfolio website developed as a virtual CV for CMPG323 Profile Development.

## Live Website

The deployed portfolio can be viewed at:

`https://coderinapurpr.github.io/virtual-cv/`

## Overview

The website presents my:

- professional profile;
- technical skills;
- featured projects;
- employment experience;
- education and certifications;
- professional contact information;
- downloadable curriculum vitae.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- GitHub Pages

## Main Features

- Responsive desktop, tablet, and mobile layouts
- Semantic HTML structure
- Dark and light colour themes
- Saved theme preference using local storage
- Responsive hamburger navigation
- Active navigation highlighting
- Scroll-reveal animations
- Terminal-inspired professional summary
- Downloadable PDF CV
- Accessible keyboard navigation
- Reduced-motion support

## Running the Website Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/CoderinaPurpr/virtual-cv.git
   ```

2. Open the project directory:

   ```bash
   cd virtual-cv
   ```

3. Open `index.html` directly in a browser, or run it with the Live Server extension in Visual Studio Code.

## Accessibility and Compatibility

The website includes:

- semantic page sections;
- descriptive navigation labels;
- keyboard focus indicators;
- accessible menu controls;
- responsive layout breakpoints;
- reduced-motion support;
- readable colour contrast in both themes.

The website is designed for current versions of major browsers, including Chrome, Edge, Firefox, and Safari.
